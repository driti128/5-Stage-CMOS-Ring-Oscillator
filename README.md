# 5-Stage CMOS Ring Oscillator

## Overview
Designed and simulated a 5-stage CMOS ring oscillator using 180nm CMOS technology in Cadence Virtuoso. The project demonstrates continuous oscillation using an odd number of inverter stages connected in a feedback loop.

## Features
- CMOS inverter-based ring oscillator
- 5-stage inverter loop
- Oscillation waveform analysis
- Capacitive load analysis
- Propagation delay study

## Tools Used
- Cadence Virtuoso
- 180nm CMOS Technology

## Working Principle
The ring oscillator generates continuous oscillations due to the odd number of inverter stages and propagation delay through each inverter.

## Applications
- Clock generation
- Delay measurement
- Timing analysis
- VLSI testing
